# Playing BBC Micro disc Elite on the BBC Master

## How to download and play the original disc version of Elite on the BBC Master

The [Elite Compendium](https://elite.bbcelite.com/elite_compendium.html) is a great way to enjoy BBC Micro Elite in combination with my other hacks - see the [download page](https://elite.bbcelite.com/elite_compendium_downloads.html) for details. Or you can play classic BBC Micro Elite on the Master by choosing one of the following:

- Play the BBC Master version of BBC Micro disc Elite [in your browser](https://bbc.xania.org/?model=Master&autoboot&disc=https://elite.bbcelite.com/versions/bbc_master_disc_elite/elite-disc-bbc-master-sth.ssd).
- Play the flicker-free BBC Master version of BBC Micro disc Elite [in your browser](https://bbc.xania.org/?model=Master&autoboot&disc=https://elite.bbcelite.com/versions/bbc_master_disc_elite/elite-disc-bbc-master-flicker-free-sth.ssd).
- Download the [BBC Master version of BBC Micro disc Elite as a disc image](https://elite.bbcelite.com/versions/bbc_master_disc_elite/elite-disc-bbc-master-sth.ssd)(or download the[flicker-free version](https://elite.bbcelite.com/versions/bbc_master_disc_elite/elite-disc-bbc-master-flicker-free-sth.ssd)). This will run in an emulator, or on a real BBC Master 128 using a device like a Gotek, and is probably the version you want (specifically, this is the[Stairway to Hell variant](https://elite.bbcelite.com/disc/releases.html#sth), which contains all the bug fixes).
- You can also download a [BBC Master version of the variant on Ian Bell's site](https://elite.bbcelite.com/versions/bbc_master_disc_elite/elite-disc-bbc-master-ib-disc.ssd)(or download the[flicker-free version](https://elite.bbcelite.com/versions/bbc_master_disc_elite/elite-disc-bbc-master-flicker-free-ib-disc.ssd)). This contains the bugs from the very first release ([see more details](https://elite.bbcelite.com/disc/releases.html#ib-disc)) and is aimed at those interested in the history of the game; you probably don't want to play this one.

To load BBC Micro disc Elite on a BBC Master, simply insert the disc image and press SHIFT-D-BREAK (for DFS), SHIFT-M-BREAK (for MMFS), SHIFT-S-BREAK (for Micro SPI) or use the *DBOOT command (for all SD card solutions); emulators tend to map BREAK to F12, so press SHIFT-D-F12. The disc will automatically load if you're playing it in a browser.

Incidentally, this hacked version only works on the BBC Master, and will not work on the BBC Micro. BBC Micro owners should, of course, play the [original disc version](https://elite.bbcelite.com/disc/) instead.


													 ---------------

						BBC Micro disc Elite on the BBC Master has had the following releases:

- 2022-12-19 - Initial release
- 2023-02-22 - Released a flicker-free version (with flicker-free ships and planets)

You can check the release for a given disc image by loading the disc and typing *TYPE README to display the credits. The build date is at the end.
