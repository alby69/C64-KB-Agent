# Playing BBC Master Elite on the BBC Micro B+

## How to download and play BBC Master Elite on the BBC Micro B+

![The Cougar in the BBC Master version of Elite](https://elite.bbcelite.com/images/master/cougar.png) 

						The [Elite Compendium](https://elite.bbcelite.com/elite_compendium.html) is a great way to enjoy BBC Master Elite on the BBC Micro B+ in combination with my other hacks - see the [download page](https://elite.bbcelite.com/elite_compendium_downloads.html) for details.

There is only one way to play BBC Master Elite on the BBC Micro B+, and that is to download a disc image. Sadly none of the web-based emulators emulate the B+, so you'll either have to install an emulator on a modern machine, or load the disc image into a real B+. Here's the link:

- Download the BBC Micro B+ version of BBC Master Elite [as a disc image](https://elite.bbcelite.com/versions/bbc_micro_b_plus_master_elite/elite-master-bbc-micro-b-plus-sng47.ssd). This will run in an emulator, or on a real BBC Micro B+ or B+128 using a device like a Gotek.

To load BBC Master Elite on a BBC Micro B+, simply insert the disc image and press SHIFT-D-BREAK (for DFS), SHIFT-M-BREAK (for MMFS), SHIFT-S-BREAK (for Micro SPI) or use the *DBOOT command (for all SD card solutions); emulators tend to map BREAK to F12, so press SHIFT-D-F12.


													 ---------------

						BBC Master Elite on the BBC Micro B+ has had the following releases:

- 2025-08-01 - Initial release
- 2025-08-30 - Fixed a minor bug in the disc menu error handling
- 2025-09-18 - Fixed the catalogue and delete options in the disc menu, which weren't working on real hardware

You can check the release for a given disc image by loading the disc and typing *TYPE README to display the credits. The build date is at the end.
