# Site history

Here's a quick history of the development of this site, with the most recent additions first:

- Apr 2026: [Flicker-free ships and planets](https://elite.bbcelite.com/hacks/flicker-free_elite.html)added to Elite Demonstration Disc
- Apr 2026: [Elite Demonstration Disc](https://elite.bbcelite.com/demo/)added
- Aug 2025: Added the [Elite Compendium Addendum for the BBC Micro B+](https://elite.bbcelite.com/hacks/elite_compendium.html)
- Aug 2025: Added [BBC Master Elite on the BBC Micro B+](https://elite.bbcelite.com/hacks/bbc_micro_b_plus_master_elite.html)
- Jul 2025: Added the [Compendium version of Acorn Electron Elite](https://elite.bbcelite.com/hacks/elite_compendium_acorn_electron.html)
- Jun 2025: Acorn Electron Elite added to [Elite over Econet](https://elite.bbcelite.com/hacks/elite_over_econet.html)
- May 2025: [Flicker-free planets](https://elite.bbcelite.com/hacks/flicker-free_elite.html)added to Apple II Elite
- Apr 2025: Archimedes Elite support added to [Elite over Econet](https://elite.bbcelite.com/hacks/elite_over_econet.html)
- Mar 2025: Flicker-free version of [Elite over Econet](https://elite.bbcelite.com/hacks/elite_over_econet.html)added
- Feb 2025: Apple II [deep dives](https://elite.bbcelite.com/deep_dives/)added, bringing the total number to 130
- Jan 2025: [Apple II version](https://elite.bbcelite.com/apple/)added
- Dec 2024: [Commodore 64 version](https://elite.bbcelite.com/c64/)added along with a number of[deep dives](https://elite.bbcelite.com/deep_dives/), bringing the total number to 120
- Nov 2024: Added Delta 14B support to all versions in the [Elite Compendium](https://elite.bbcelite.com/hacks/elite_compendium.html)
- Oct 2024: Created buildable repositories for the [Commodore 64](https://github.com/markmoxon/elite-source-code-commodore-64)and[Apple II](https://github.com/markmoxon/elite-source-code-apple-ii)source code
- Sep 2024: Released [Elite 3D](https://elite.bbcelite.com/hacks/elite_3d.html)
- Jul 2024: Added enhanced features to all versions in the [Elite Compendium](https://elite.bbcelite.com/hacks/elite_compendium.html)
- Jun 2024: Moved the Elite site to [elite.bbcelite.com](https://elite.bbcelite.com/)and converted[bbcelite.com](https://www.bbcelite.com/)into a parent site for all my software archaeology projects
- May 2024: Added a [multiplayer scoreboard to Elite over Econet](https://elite.bbcelite.com/hacks/elite_over_econet_scoreboard.html)
- Apr 2024: Added the [sideways RAM variant of BBC Micro disc Elite](https://elite.bbcelite.com/disc/releases.html#sideways-ram)
- Mar 2024: Added [to-do list](https://elite.bbcelite.com/about_site/to-do_list.html)
- Mar 2024: Big improvements to the cross-linking of workspace and configuration variables
- Jan 2024: [Elite Compendium](https://elite.bbcelite.com/hacks/elite_compendium.html)and[Elite over Econet](https://elite.bbcelite.com/hacks/elite_over_econet.html)added
- Dec 2023: [25 deep dives](https://elite.bbcelite.com/deep_dives/)added to accompany the NES version
- Oct 2023: [NES version](https://elite.bbcelite.com/nes/)added
- Oct 2023: Fully documented [source code for the NES version](https://github.com/markmoxon/elite-source-code-nes)released on GitHub
- Jun 2023: Extended the deep dive on [drawing Saturn on the loading screen](https://elite.bbcelite.com/deep_dives/drawing_saturn_on_the_loading_screen.html)
- May 2023: Added a deep dive on the [space station safe zone](https://elite.bbcelite.com/deep_dives/the_space_station_safe_zone.html)
- Apr 2023: Repository created with [buildable but uncommented NES disassembly](https://github.com/markmoxon/elite-source-code-nes/commit/a162cbb7fed03f7eeca362992529228cca5790a1)
- Apr 2023: Improved the [code statistics](https://elite.bbcelite.com/cassette/articles/source_code_statistics.html)to cover data blocks and macros
- Mar 2023: Added a new deep dive on [drawing ellipses](https://elite.bbcelite.com/deep_dives/drawing_ellipses.html)and extended the deep dives on[meridians](https://elite.bbcelite.com/deep_dives/drawing_meridians_and_equators.html)and[craters](https://elite.bbcelite.com/deep_dives/drawing_craters.html)
- Feb 2023: [Flicker-free planets](https://elite.bbcelite.com/hacks/flicker-free_elite_technical_information.html)added to Teletext Elite, and Elite on the BBC Master and 6502SP
- Feb 2023: [BBC Micro Elite... with music!](https://elite.bbcelite.com/hacks/bbc_elite_with_music.html)released for the BBC Micro with 16K of sideways RAM
- Dec 2022: [BBC Micro disc Elite](https://elite.bbcelite.com/hacks/bbc_master_disc_elite.html)released for the BBC Master
- Nov 2022: [Teletext Elite](https://elite.bbcelite.com/hacks/teletext_elite.html)released for the BBC Micro and BBC Master
- Oct 2022: [Elite Universe Editor](https://elite.bbcelite.com/hacks/elite_universe_editor.html)released for the BBC Micro and BBC Master
- Sep 2022: [Flicker-free Elite](https://elite.bbcelite.com/hacks/flicker-free_elite.html)released for the Commodore 64 and Plus/4
- Aug 2021: [Flicker-free Elite](https://elite.bbcelite.com/hacks/flicker-free_elite.html)released for the BBC Micro and Acorn Electron
- Aug 2021: [Elite-A](https://elite.bbcelite.com/elite-a/)and associated deep dives added
- May 2021: [Executive version](https://elite.bbcelite.com/6502sp/releases.html#executive)added
- Apr 2021: [BBC Master version](https://elite.bbcelite.com/master/)and associated deep dives added
- Apr 2021: [Acorn Electron version](https://elite.bbcelite.com/electron/)and associated deep dives added

- Jan 2021: [6502 Second Processor version](https://elite.bbcelite.com/6502sp/)and associated deep dives added
- Dec 2020: More [deep dives](https://elite.bbcelite.com/deep_dives/)and[code statistics](https://elite.bbcelite.com/cassette/articles/source_code_statistics.html)added
- Nov 2020: Improved [cross-references](https://elite.bbcelite.com/cassette/articles/source_code_cross-references.html)and links within the code

- Sep 2020: Website launched with [BBC Micro cassette version](https://elite.bbcelite.com/cassette/)
