                # Memory configuration

This section describes how to do various kinds of memory configuration on the C64.

- [Quick VIC-II screen setup](https://codebase.c64.org/doku.php?id=base:quick_vicii_screen_setup)- Code snippet to easily set the screen/font/bitmap address configuration
- [VIC memory organizing](https://codebase.c64.org/doku.php?id=base:vicii_memory_organizing)- Selecting VIC bank, where to put sprites, etc.
- [Memory Management](https://codebase.c64.org/doku.php?id=base:memmanage)- a part of c64doc by John West and Marko Mäkelä
- [Memory Management](https://codebase.c64.org/doku.php?id=base:memory_management)- Switching on/off BASIC, KERNAL, CHAR ROM by setting $01.
- [RAM beneath $00 and $01](https://codebase.c64.org/doku.php?id=base:ram_beneath_00_and_01)- For the nerd in you.
