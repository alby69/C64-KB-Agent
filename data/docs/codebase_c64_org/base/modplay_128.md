                # Modplay 128

This page provides a copy of the source code to Modplay 128 v1.0. This freeware program is a player for standard Protracker or Noisetracker MOD.* files, for the C128. This program's look/feel was based on Mark Dickenson's Stereo SID Player v10.3, and features a simple playlist editor, loadable drivers, and more. This program uses stock KERNAL routines, so it should work with most any storage device. Unlike the C64 version, this one does NOT support the SuperCPU. Requires only a stock Commodore 128, an REU, an 80-column display, and a drive big enough to hold your music.
