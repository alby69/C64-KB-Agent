# PDS (Programmers Development System)

PDS is a cross development system from the 80's that is hardly in use anymore. Still, there are some old sources around, and sometimes people want to convert these sources to something which can be compiled with modern cross assemblers. This is the topic of the present page, as well as general info on this system.
