# MIDI on the C64

A number of MIDI interfaces exist for the C64. Most of them take the form of cartridges.

# Specs

- [C64 MIDI Interfaces](https://codebase.c64.org/doku.php?id=base:c64_midi_interfaces)- Description of C64 Midi interfaces and registers, by GRG.
- [C64 MIDI Data](https://codebase.c64.org/doku.php?id=base:c64_midi_data)- Short Description of data, by GRG.
